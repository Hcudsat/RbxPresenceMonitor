RbxPresence Monitor

Cloud-connected Roblox activity tracker for Discord

About

RbxPresence Monitor allows you to monitor any Roblox user’s online status and send updates directly to a Discord channel. Built with a cloud backend for reliability and privacy.

Features

  • • Real-time Roblox activity tracking
  • • Secure Discord webhook integration
  • • Cloud-hosted background monitoring
  • • Minimalist UI and transparent design
