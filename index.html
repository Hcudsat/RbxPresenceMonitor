RbxPresence Monitor

Monitor Roblox user activity from your browser

About

RbxPresence Monitor is a Chrome Extension that allows you to monitor any Roblox user's online or in-game status and receive instant Discord notifications. The monitoring runs on a secure cloud backend, ensuring real-time updates and high reliability.

Features

  • • Real-time Roblox presence tracking
  • • Discord webhook integration
  • • Cloud-hosted monitoring engine
  • • Lightweight browser integration
