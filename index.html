RbxPresence Monitor

Monitor Roblox activity. Stay connected.

About

RbxPresence Monitor is a browser extension that connects Roblox with Discord. It tracks user activity (online, offline, in-game) in real-time and sends updates through your custom webhook.

Features

  • Real-time Roblox status tracking
  • Secure Discord webhook integration
  • Cloud-based backend for high uptime
  • Fast, lightweight Chrome extension
