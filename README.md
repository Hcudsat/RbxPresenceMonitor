# Roblox Presence Notifier

A simple cloud-hosted Python app that monitors a Roblox user's online status and sends notifications to a Discord channel.

---

## Features
-  Monitors online / offline / playing state via Roblox Presence API  
-  Runs continuously on Render or Railway  
-  Sends status updates to Discord Webhook  
-  Includes Flask health-check endpoint (`/health`)
