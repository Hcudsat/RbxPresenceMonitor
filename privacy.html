Privacy Policy – RbxPresence Monitor

Last updated: October 13, 2025

1. Overview

RbxPresence Monitor (“the Extension”) is a Chrome Extension developed by an independent developer. The Extension allows users to monitor the online status of specific Roblox users and receive updates through a Discord webhook.

2. Information We Collect

  • User Input: Roblox User ID and Discord Webhook URL entered by the user are stored locally in the browser (via chrome.storage.sync). These values are not transmitted to any third-party server except the webhook you provide.
  • API Requests: When monitoring starts, the Extension communicates with an API hosted on Render to fetch Roblox presence data and send notifications to your provided Discord webhook.

3. Data Retention

No personal data is permanently stored on our servers. Temporary diagnostic logs may contain Roblox User IDs and timestamps but are automatically deleted within 24 hours.

4. Third-Party Services

This Extension interacts with the following third-party APIs:

  • Roblox Presence API (for user activity data)
  • Discord Webhook API (for notifications)

5. User Rights

Users can clear all saved data at any time by removing the Extension or clearing Chrome storage. This Extension does not collect or share personally identifiable information.

6. Changes to This Policy

This Privacy Policy may be updated at any time. Updates will be reflected in this document with a revised “Last updated” date.

7. Contact

For questions or concerns, please contact the developer via GitHub: https://github.com/Hcudsat
